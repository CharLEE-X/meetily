# Legacy Backend Agent Instructions

This directory is an archived Python/FastAPI, Docker, and standalone whisper-server backend from older Meetily releases.

## Rules

- Do not use this directory for current app development, new installs, production deployment, or supported security review.
- Do not add new supported product behavior here.
- Do not treat the old FastAPI CORS or unauthenticated behavior as the current app security model.
- Read `README.md` before using these files for migration or historical comparison.

Current supported backend behavior lives in `../frontend/src-tauri/`.
